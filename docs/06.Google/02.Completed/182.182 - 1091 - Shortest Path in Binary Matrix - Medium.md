## Solution: BFS
伪代码
```
void BFS()
{
    判断边界条件，是否能直接返回结果的：如果 grid[0][0]=1, 或者grid[n-1][n-1]=1,
                                  那肯定不能从左上走到右下，返回 -1

    定义队列：一个list，里面每一个元素时一个tuple [(x, y)]
           初始化为 [(0, 0)]
    
    定义备忘录，用于记录已经访问的位置： 本题不需要，直接把visited的格子改成1

    将起始位置加入到队列中，同时更新备忘录: 本题没有备忘录，所以先把grid[0][0]变成 = 1

    初始化step = 1

    while (队列不为空) {
        获取当前队列中的元素个数。
        for (元素个数) {
            取出一个位置节点: x, y = q.popleft()
            判断是否到达终点位置, 如果是，返回 step.
            获取它对应的下一个所有的节点: 即当前点周围的八个位置
            条件判断，过滤掉不符合条件的位置: 即判断这些点是否 in_area 且值为0
            新位置重新加入队列。
            visited.add()： 本题没有visited，把新的位置标记成1
        }
        step += 1
    }
    返回 (如果是search，则可能是返回-1)

}
```

```python
from collections import deque
class Solution:
    def shortestPathBinaryMatrix(self, grid):
        n = len(grid)
        if grid[0][0] or grid[n-1][n-1]:
            return -1
        q = deque([(0, 0)])
        grid[0][0] = 1
        step = 1

        def in_area(x, y):
            return 0 <= x <= n-1 and 0 <= y <= n-1

        while q:
            size = len(q)
            for _ in range(size):
                x, y = q.popleft()
                if x == y == n - 1:
                    return step
                for next_x, next_y in [(x-1, y-1), (x-1, y), (x-1, y+1),
                                      (x, y-1),             (x, y+1),
                                      (x+1, y-1), (x+1, y), (x+1, y+1)]:
                    if in_area(next_x, next_y) and grid[next_x][next_y] == 0:
                        q.append((next_x, next_y))
                        grid[next_x][next_y] = 1
            step += 1
        return -1
```
