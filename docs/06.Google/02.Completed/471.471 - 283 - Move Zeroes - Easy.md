套用quick select中partition的方法
```python
class Solution:
    def moveZeroes(self, nums):
        idx, marker = 0, 0
        while idx < len(nums):
            if nums[idx] != 0:
                nums[idx], nums[marker] = nums[marker], nums[idx]
                marker += 1
            idx += 1
```
