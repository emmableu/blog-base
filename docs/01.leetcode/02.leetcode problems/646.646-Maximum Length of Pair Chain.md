## Solution: greedy (O(n lg(n)))
```python
class Solution:
    def findLongestChain(self, pairs):
        # 按end升序排序
        pairs.sort(key = lambda x: x[1]) 
        # 至少有一个区间不相交
        cnt = 1
        # 排序后， 第一个区间就是x
        x_end = pairs[0][1]
        for pair in pairs:
            start = pair[0]
            if (start > x_end):
                # 找到了下一个选择的区间
                cnt += 1
                x_end = pair[1]
        return cnt
```
