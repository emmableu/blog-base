## solution: 寻找战舰头
  由于战舰只能水平或者垂直放置，所以战舰头的左边和上边一定不是 X。

```python
class Solution(object):
    def countBattleships(self, board):
        if len(board) == 0: return 0
        m, n = len(board), len(board[0])
        count = 0
        for i in range(m):
            for j in range(n):
                if board[i][j] == 'X' \
                and (i == 0 or board[i-1][j] == '.') \
                and (j == 0 or board[i][j-1] == '.'):
                    count += 1
        return count
```
